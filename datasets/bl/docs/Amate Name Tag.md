# Amate Name Tag

[Aside block] Amate Name Tag
----------------------------

### Type

Other

### Renewable

Yes

### Stackable

Yes (64)

### Compostable

No

### ID

bl\_nametag

**Amate Name Tags** are a craftable item that are the betweenlands equivalent of the Nametag.

Crafting Recipe
-----------------

| Result | Ingredients | Recipe |
| --- | --- | --- |
| Amate Nametag | Reed Rope + Amate Paper |  |

Usage
-------

Shift right click with the name tag in hand to name it. Click save to save the name. The name can be overwritten with the same steps.
