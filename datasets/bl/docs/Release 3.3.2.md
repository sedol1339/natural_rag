# Release 3.3.2

**Release 3.3.2** was an update to the Betweenlands for Minecraft 1.12.2 that was released on March 9, 2018. It added various changes and bug fixes.

Changelog
---------

### Changes

* Reduced Purifier capacity to 4 buckets, but also reduced water cost equally
* Reduced number of rendered items in Animator and Purifier
* Decreased Smoll Sludge health to 14
* Sludges now only hide in dirt type blocks
* Added small XP cost to activate Ring of Ascent
* Swamp Talisman can now create small portal in Portal Frame

### Fixes

* Fixed Repeller liquid not glowing
* Fixed some renderers causing mipmap to flicker when looking at Draconic Evolution's Pig Soul item
* Fixed Dark Druid Altar causing a rendering bug when in inventory
* Fixed recipe overrides causing recipe book crash
* Fixed full bark Weedwood Log not grindable in Pestle & Mortar
* Fixed Spike Traps dropping and breaking unbreakable blocks
* Fixed Ring of Ascent being active in creative and spectator mode
* Fixed Sludge Slicer not killing Smoll Sludge
* Fixed Portal Frame not dropping any item
