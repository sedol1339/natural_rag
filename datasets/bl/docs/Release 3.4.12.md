# Release 3.4.12

**Release 3.4.12** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on September 30, 2019. It added back the Volarkite, as well as a config option and a couple of bug fixes.

Changelog
---------

### Additions

* Added Volarkite
* Added shader compat option for e.g. LadyLib/Dissolution (see Shader Priority in config)

### Bug Fixes

* Fixed Weedwood Bow not properly detecting infinite arrows
* Fixed Ring of Power working even when fully depleted
