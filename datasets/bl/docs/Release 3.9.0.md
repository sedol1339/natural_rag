# Release 3.9.0

Credit to Pasta for the image

**Release 3.9.0** was a major update to the Betweenlands for Minecraft 1.12.2 that was released on December 29, 2022. It was known as the **Moth 'n Broth 2: Dye Another Day** update as it added a new farming system for Silk, a new brewing system in the Steeping Pot, Dyes and dyed blocks, Drinkable Brews, and a host of new building and decorative blocks, alongside several other additions, changes, and bug fixes.

Changelog
---------

### Additions

* New brewing mechanics
* New farming mechanic (silk)
* Added Steeping Pot, for making new drinks and dyes
* Added Water Filter
* Added Filtered Silt Glass Jars
* Added Clean Water
* Added Fish Oil
* Added 11 new Drinkable Brews + fluids, some of which have unique effects
* Added 16 Dyes + fluids
* Added dyed variants of Mud Brick Shingles, Filtered Silt Glass, and Reed Mats
* Lurker Skin Pouches can now be dyed
* Added a new textile block called Samite + dyed variants
* Added Samite Canvas Panels + dyed variants
* Added Item Frames + dyed variants + glowing variants + invisible variant
* Gave Infestations a new drop: Pheromone Thorax
* Weedwood Bushes can now be infested by silk worms.
* Added Silk Grubs, Silk Grub Cocoons and Silk Thread
* Added Silk Bundle for Steeping Pot drinks and dyes
* Added Silk Mask, a new 'armour' that keeps you safe from spores
* Added Grub Hub, a new block that helps your silk farm
* Added Moth House, a more efficient way of silk harvesting
* Added 6 Treated Plank variants and corresponding blocks
* Added 6 Treated Door and Trapdoor variants
* Added Rotten Plank Door and Trapdoor
* Added Lattice and Fine Lattice
* Added Ancient Remnant Block
* Added 14 new Advancements

### Changes

* Weedwood Fishing Rods now use Silk Thread instead of Reed Rope
* Lurker Skin Pouches now use Silk Thread instead of Reed Rope
* Made Urchin Spikes a damage source
* Removed all armour benefits from Spirit Tree Face Masks and made them just decorative
* Made Small Spirit Tree Faces ignore players wearing a Spirit Tree Face Mask
* Anadia Remains can now be crafted
* Changed Nettle Soup to be a Drinkable Brew (updating an existing world will show item is missing, but this is no problem)
* Updated Russian translations (big thanks to Avale)
* Updated French translations (big thanks to TheSilverplier)

### Bug Fixes

* Fixed Harlequin Toad & Chiromaw naming
* Removed enchanting from various items
* Added defensive code to stop other mods crashing Freshwater Urchins by messing with their attack code
* Fixed Spike Trap derp
