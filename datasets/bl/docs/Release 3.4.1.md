# Release 3.4.1

**Release 3.4.1** was an update to the Betweenlands for Minecraft 1.12.2 that was released on November 5, 2018. It added some new sounds, translations and bug fixes.

Contents
--------

* 1 Changelog
  + 1.1 Additions
  + 1.2 Changes
  + 1.3 Bug Fixes

Changelog
---------

### Additions

* Added Traditional Chinese translation (zh\_TW). Thanks mcBegins2Snow!
* Added JEI support for some recipes added in the last update

### Changes

* Improved Boulder Sprite sounds and added idle sounds

### Bug Fixes

* Fixed small mistake in Herblore Book
* Fixed fire/lava type damage always reduced
* Fixed crash with JEI
* Fixed crash when Algae is placed in a Mud Brick Flower Pot
