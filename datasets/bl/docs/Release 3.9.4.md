# Release 3.9.4

**Release 3.9.4** was a small update to the **Betweenlands** for Minecraft 1.12.2 that was released on January 21, 2023. It was originally released as **Release 3.9.2**, which was later archived due to a critical bug, and **3.9.3** was never released due to an additional bug that was fixed last minute. In total, the update added a couple new quality-of-life items, as well as a few changes and bug fixes.

Changelog
---------

### 3.9.4 Changelog

#### Bug Fixes

* Fixed the Purifier stack size on output (really this time)

### 3.9.3 Changelog

#### Bug Fixes

* Fixed Clutch of Pheromone Thoraxes eating any block it was used on
* Removed Shadow and Mist blocks from silk touch

### 3.9.2 Changelog

#### Additions and Changes

* Added Silky Pebble item
* Added Clutch of Pheromone Thoraxes item
* Changed Moss generation to be more friendly to the world
* Made Pheromone Thorax stackable
* Plants can now be planted on Purified Swamp Dirt
* Changed Weedwood Fishing Rod line colour to match new silk colour

#### Bug Fixes

* Fixed Moth House Drops
* Fixed Purifier honouring stack size on output
* Fixed typo in lang for Treated Rubber Tree Plank Fences
* Fixed Drinkable Brews effects that were missing
* Fixed Silk Bundle GUI and Container to close when dropped
* Removed pointless blocks from creative and JEI
