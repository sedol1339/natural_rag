Valonite Ore in-game

Obtaining
-----------

### Generation
Valonite Ore generates rather rarely underground between Y-layers 45 and 0 within Pitstone, in the Lake Cavern layer.

### Drops
Valonite Ore can drop rarely from Boulder Sprites.

Drops
-------

Valonite Ore drops a Valonite Shard and 5-12 XP when broken. The block itself must be broken with a Silk Touch tool to be collected.
