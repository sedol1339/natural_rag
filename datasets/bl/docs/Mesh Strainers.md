# Mesh Strainers

Mesh Strainers are used to filter Swamp Water or Stagnant Water in a Water Filter.

[Aside block] Mesh Strainers
----------------------------

* Moss
* Silk

### Type

Other

### Durability

Moss Mesh Strainer: 400  
Silk Mesh Strainer: 2000

### Tooltip

Used in: Water Filter

### Renewable

Yes

### Stackable

No

### Compostable

No

### ID

moss\_filter  
silk\_filter

Crafting Recipe
-----------------

| Result | Ingredients | Recipe |
| --- | --- | --- |
| Moss Mesh Strainer | Weedwood Stick (4) + Moss |  |
| Silk Mesh Strainer | Weedwood Stick (4) + Silk Thread |  |

Usage
-------

Right-click a Water Filter with the mesh to add it to the filter. Shift-right click or break the filter to retrieve the mesh.
