# Release 3.3.6

**Release 3.3.6** was an update to the Betweenlands for Minecraft 1.12.2 that was released on April 4, 2018. It added a few changes and bug fixes.

Changelog
---------

### Changes

* Improved respawn position selection
* Grinding Limestone now gives 3 Limestone Flux instead of 1
* Mortar is now activated with left-click and GUI is opened with right-click
* Compost Bin is now opened/closed with left-click
* Readded Compost Bin composting particles

### Fixes

* Fixed BL fluid Buckets added as fuel with CraftTweaker not matching, making empty buckets burnable
* Fixed Rings XP dupe bug
* Fixed Life Crystals invisible when Optifine installed
* Fixed Tar Minion portal dupe bug
