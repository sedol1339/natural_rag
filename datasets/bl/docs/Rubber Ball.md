**Rubber Balls** are items primarily used in making boots and vials.

They can be processed directly from a weedwood bucket, or from a **Syrmorite Bucket with Solid Rubber**.

### Smelting Recipes
Smelting a Weedwood Bucket with Rubber will directly produce 3 Rubber Balls and destroy the bucket in the process. Smelting a Syrmorite Bucket with Rubber will instead produce a Syrmorite Bucket with Solid Rubber, which can be reused once emptied.

| Result | Ingredients | Recipe |
| --- | --- | --- |
| **Rubber Ball** (3) + 0.5 XP | Weedwood Bucket (Rubber) + Any fuel |  |
| **Syrmorite Bucket with Solid Rubber** + 0.5 XP | Syrmorite Bucket (Rubber) + Any fuel |  |

### Throwing Infusions
A Rubber Ball has a chance to be left behind upon throwing any Infusion.

Properties
------------

### Emptying
Right-clicking a Syrmorite Bucket with Solid Rubber will yield 3 Rubber Balls and leave behind an empty Syrmorite Bucket.
