# Release 3.6.2

**Release 3.6.2** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on July 25, 2020. It made major adjustments to Greebling Riders and the Draeton, and fixed some more bugs.

Changelog
---------

### Additions and Changes

* Greebling Riders now have a revenge AI
* Greebling Riders now no longer target Draetons or their pullers when on the ground
* Draeton now has 33% chance to get leakages from projectiles instead of 100%
* Draeton leakages are now temporary and last for 5 minutes
* Draeton fuel (Bottled Shallowbreath) now lasts twice as long
* Draeton now floats on water
* Increased number of Lurker Skin Patches obtained from its recipe to 4
* *SeriousCreeper* added configs for ore vein size
* Added sound when removing Rubber from Syrmorite Bucket

### Bug Fixes

* Fixed Greebling Riders hanging from ceilings
* Fixed Tamed Chiromaws targetting each other
* Fixed some structure loot pools not working until reloaded
* Fixed infinite Tamed Chiromaw double jumps
* Fixed Greataxes breaking blocks on rightclick swing
* Fixed Ancient Greatsword extended reach only working for default keybind
