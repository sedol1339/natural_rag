# Release 3.3.9

**Release 3.3.9** was an update to the Betweenlands for Minecraft 1.12.2 that was released on June 20, 2018. It added improved visuals and variants to Rifts, as well as some other changes and bug fixes.

Changelog
---------

### Additions and Changes

* Added new and improved sky Rift variants
* Added config to create custom portals
* Added config to specify unsafe biomes for portal to avoid
* Added config to let players start in The Betweenlands
* Added config to let players always respawn in The Betweenlands instead of The Overworld
* Added config item blacklist for Lurker Skin Pouch
* Added Tomb Many Graves 2 compatibility for equipped items
* Added event textures to Poison Ivy and Rubber Tree Log
* Slightly improved performance of equipped items
* Dark Druid Spawner under Dark Druid Altar can now be reactivated by putting 4 saplings in the Druid Altar
* Tar Minions no longer drown in Tar
* Updated zh\_CN localization

### Bug Fixes

* Fixed water invisible when in Weedwood Rowboat with Galacticraft installed
* Fixed Gecko Cage having a cooldown after Gecko dies
* Fixed a crash when certain items were rendered in Weedwood Crafting Table
* Fixed Dynamic Trees saplings not detected as sapling when trying to create portal
* Fixed Animator losing items when unloaded
* Fixed Block of Compost not giving 9 Compost when uncrafting
* Fixed Middle Gem Blocks, Coral Blocks and Angry Betweenstone not glowing
* Fixed apparent crash when entering The Betweenlands
* Fixed Cragrock Tower areas sometimes overlapping in rare cases
* Fixed incorrect description of the torch black-/whitelist config
