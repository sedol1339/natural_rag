# Release 3.5.4

**Release 3.5.4** was a very small update to the Betweenlands for Minecraft 1.12.2 that was released on December 1, 2019. It improved some AI, added a new config option and fixed a lang inconsistency.

Changelog
---------

### Additions

* Added swimming AI to recruited mobs
* Added config blacklist for Moving Walls block breaking

### Bug Fixes

* Fixed US lang spelling for Fashionable Warrior Advancement
