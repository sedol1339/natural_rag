# Beta 2.0.1

**Beta 2.0.1** was an update to the **Betweenlands** for Minecraft 1.10.2 that was released on March 5, 2017. It added some missing blocks from the 1.7.10 version, some new blocks, missing recipes and bug fixes.

Changelog
---------

* Fixed occasional ConcurrentModificationException caused by Mud Brick Flower Pot and both Rubber Taps
* Fixed BL saplings crashing the game when growing
* Fixed Nettle Soup using up two items at once
* Fixed crash caused by Basilisk Arrow
* Fixed orange Dentrothyst Shield recipes not working
* Fixed Weeping Blue Petals not curing decay
* Fixed Reed Rope requirement missing when placing Rubber Taps
* Fixed Weedwood Bow not corroding
* Fixed crash during initialization with several other mods
* Fixed Decay and Ring of Recruitment renderer occasionally causing a lot of lag
* Removed food sickness from Ball of Sap
* Added missing recipes for: Syrmorite Bucket, Polished Limestone Slabs/Stairs, Weedwood Log Fences/Fence Gates, Rubber Block
* Added Polished Limestone Wall
* Updated Walkway and Weedwood Chip Path
* Updated Thatch, Thatch Slab and Thatch Roof
* Further performance improvements to world gen
* JEI integration mostly implemented
