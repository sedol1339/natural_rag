# Release 3.3.4

**Release 3.3.4** was an update to the Betweenlands for Minecraft 1.12.2 that was released on March 18, 2018. It added some improved compatibility and performance and some bug fixes.

Changelog
---------

### Changes

* Improved Optifine compatibility
* Improved Shallowbreath performance
* Improved world/chunk data file IO performance

### Fixes

* Fixed Octine Sword causing crash in some circumstances when no hand is active
* Fixed shader sky not completely fading out
* Fixed some localization entries
* Fixed Weedwood Crafting Table result not updating in all inventories causing dupe bug
* Fixed Lurker Skin Pouch not closing when item is removed from inventory causing dupe bug
* Fixed Herblore Book crash when opening in off-hand
* Fixed "Cling Bling" Advancement triggered by any equippable
