# Alpha 3.0.0

*This is a snapshot of the upcoming 1.12.2 update. This is not the final version! Changes are to be expected and some things may not be fully implemented yet.*

**Alpha 3.0.0** was a major snapshot update to the **Betweenlands** that brought the mod to Minecraft 1.12.2 and was released on December 4, 2017. It mostly re-implemented Herblore, added unique abilities to some shields, and implemented proper lighting.

Changelog
---------

* Updated to Minecraft 1.12.2
* The Betweenlands now has proper lighting
* Mostly re-implemented Herblore and all parts related/linked to it (Dreadful Peat Mummy, Aspectrus Plant, Repeller, decay regeneration, etc.)
* Re-implemented Puddles during Heavy Rain
* Some shields now have special abilities
* Middle Gems now provide both offensive and defensive bonuses, and have tooltips describing them
* Implemented menu music and added new music tracks
* Added world location to Underground Ruins
* Buckets now have Rubber Tap tooltips
* Rubber Taps now visibly pour so that when the tap is full can be gauged
* Re-added Mossy and Cracked Limestone Bricks with recipes
* Completely retextured Algae in all ways possible
* Renamed Gas Cloud to Shallowbreath
* Decreased Peat Mummy damage
* Tweaked flying mob AI (Fire Flies, Chiromaws, etc.)
* Increased Food Sickness limits
* Tweaked some sounds
* Re-enabled Animator custom recipes
* Removed Stagnant Water glow
* Fixed foot Leeches
* Fixed rendering of underwater plants
