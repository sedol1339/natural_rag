# Filtering

*Not to be confused with Water Filtering.*

The following is a list of all recipes that utilize a Crab Pot Filter.

Every recipe requires Anadia Remains and either a Silt Crab or Bubbler Crab contained in a Crab Pot above the Crab Pot Filter.

For more information on filtering, view the Crab Pot Filter page.
