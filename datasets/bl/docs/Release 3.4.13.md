# Release 3.4.13

**Release 3.4.13** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on October 21, 2019. It improved Volarkite functionality and fixed a couple of bugs.

Changelog
---------

### Changes

* Improved Volarkite up/downdrafts by making them last longer

### Bug Fixes

* Fixed shader sky and Shallowbreath textures not rendering
* Fixed Lurker Skin Pouch being repairable, causing it to lose slots when repaired
