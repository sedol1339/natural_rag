# Release 3.5.8

**Release 3.5.8** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on April 1, 2020. It slightly optimized cave generation and fixed some bugs.

Changelog
---------

### Changes

* Added fuel value to Weedwood Stick
* Small cave gen optimization by Kojoley, thanks.
* Updated zh\_CN localization, thanks mcBegins2Snow.e

Bug Fixes
---------

* Fixed Dungeon Door rune combination skipping every second symbol when cycling
* Fixed dataparameter using incorrect class
* Fixed Dual Sulfur Furnace losing inventory when inserting multiple smeltable items at once
* Fixed edge plant placement crash
* Fixed Beam Origin block missing particle texture
