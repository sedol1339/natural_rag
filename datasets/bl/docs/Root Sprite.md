Description
-------------

Root Sprites run away when a player get too close, displaying leaf particles.

Root Sprites are attracted to Sporelings.

Trivia
--------

* The Root Sprite is one of three sprite mobs in the game, the other two being the Ash Sprite and the Boulder Sprite.
* The Root Sprite is currently the only sprite mob in the game that is passive, and the only one to actively avoid the player.
  + The Root Sprite is also currently the only sprite mob to spawn on the surface.
