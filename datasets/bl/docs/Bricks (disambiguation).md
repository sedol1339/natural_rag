# Bricks (disambiguation)

"Bricks" refers to many different blocks and items. This disambiguation page lists all brick block and item variants added in the mod.

Blocks
--------

### **Manufactured**
* Betweenstone Bricks: Decorative brick blocks for Betweenstone.
* Cragrock Bricks: Decorative brick blocks for Cragrock.
* Limestone Bricks: Decorative brick blocks for Limestone.
* Pitstone Bricks: Decorative brick blocks for Pitstone.
* Mud Bricks: Decorative brown brick blocks.
* Carved Mud Bricks: Decorative carved mud-based blocks.
* Mud Brick Shingles: Decorative brown roof tile blocks.
* Scabyst Bricks: Decorative brick blocks made out of Scabyst.
* Mud Brick Flower Pot: A Betweenlands variant of the Flower Pot that can also be used as a candle.
* Slabs: Decorative slabs for the bricks listed above.
* Stairs: Decorative stairs for the bricks listed above.
* Walls: Decorative walls for the bricks listed above.
* Mud Brick Shingle Roof: Decorative stair-like blocks for Mud Brick Shingles.

### **Generated**

* Mud Brick Spike Trap: Blocks that house deadly spikes, found in the Sludgeon.
* Runged Mud Bricks: Brown brick blocks that can be climbed on, found in the Sludgeon.
