# Release 3.3.10

**Release 3.3.10** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on June 28, 2018. It added updated translations and changes to the Infuser, as well as a few bug fixes.

Changelog
---------

### Additions and Changes

* Infuser now plays an animation and sound when creating an "invalid" Infusion
* Updated zh\_CN and ru\_RU translations

### Bug Fixes

* Fixed Buckets with Rubber filled from modded tanks not smeltable
* Fixed equipment items visible in spectator mode and overlapping with offhand slot when primary hand is set to left
* Fixed Infuser losing contents when reloading world
