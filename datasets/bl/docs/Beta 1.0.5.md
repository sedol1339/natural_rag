# Beta 1.0.5

**Beta 1.0.5** was an update to the **Betweenlands** for Minecraft 1.7.10 that was released on June 19, 2016. It added a new mob and German localization, among other tweaks and bug fixes.

Changelog
---------

* Added Gas Cloud mob, spawns in Marsh biomes and from evaporated infusions
* Added German localization (except Herblore Book)
* Improved fog effect if graphics are set to "fancy"
* Caving Rope is now directly added to the inventory when collected
* Increased default Druid Circle spawn rate (make sure to reset the config or update it yourself!)
* Fixed increasing fall damage when using the Lightweight Draught to walk on water
* Fixed Aspect Vials crashing when a player leaves and certain other mods are installed
* Fixed shaders sometimes not working properly
