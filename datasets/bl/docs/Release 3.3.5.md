# Release 3.3.5

**Release 3.3.5** was an update to the Betweenlands for Minecraft 1.12.2 that was released on March 31, 2018. It added various changes and bug fixes.

Changelog
---------

### Changes

* Added zh\_CN translation, thanks to RisingInIris2017 and DL\_Slime!
* Added more config options regarding Rotten Food
* Adjusted mob spawn numbers to be more consistent between SP and MP
* Improved shader compatibility with mods that render custom overlays
* Wights now only possess Swamp Hags when having an attack target
* Wights now only spawn Volatile Souls if its attack target is in line of sight and isn't wearing a Skull Mask
* Cave Moss can now grow when replanted
* Purifier and Infuser will now fill up with rain water in the Betweenlands
* Underwater plants can now be farmed with composted soil like any other plant

### Fixes

* Fixed incorrect Aspect distilling loss and slightly adjusted static Aspect amounts to compensate for fix
* Fixed Alembic being able to overfill vials
* Fixed Syrmorite Hoppers losing inventory when activated or deactivated with redstone
* Fixed AttackDamageHandler crash
* Fixed silk touched Portal Frame causing crash
* Fixed naturally generated Hangers growing
* Fixed players not always respawning near portal when dying without a respawn point
* Fixed decay not resetting health to correct maximum value
* Fixed 'Dimwitted' Advancement not triggering for all torches
* Potential fix for players instantly losing connection when joining a server
