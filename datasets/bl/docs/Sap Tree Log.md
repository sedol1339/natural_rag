Sap Tree Logs ingame

**Sap Tree Logs** are the log blocks of Sap Trees. They have a normal rotatable variant and a variant with bark on all sides.

Collection
------------

When harvested normally, Sap Tree Logs drop 1-2 Balls of Sap. These items are extremely important to survival for their ability to restore decay.

Sap Tree Logs themselves must be harvested with a Silk Touch tool to be collected.

Fuel
------

Either variant of Sap Tree Log can be used as fuel for the Sulfur Furnace or Dual Sulfur Furnace, smelting 1.5 items per log.

Events
--------

Placed Sap Tree Logs will take on an alternative texture during the Spook seasonal event.
