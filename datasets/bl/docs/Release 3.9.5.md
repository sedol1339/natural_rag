# Release 3.9.5

Eternal Melodies

**Release 3.9.5** was a very small update to the **Betweenlands** for Minecraft 1.12.2 that was released on January 22, 2023. It implemented the API used to support the new Betweenlands Music addon mod, Eternal Melodies, which adds new tracks into the mod by **Rotch Gwylt**.
