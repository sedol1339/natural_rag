# Release 3.5.3

**Release 3.5.3** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on November 21, 2019. It added various improvements to items and a new Advancement, and fixed several bugs.

Changelog
---------

### Additions and Changes

* Valonite Greataxe now has twice the durability
* Improved Greataxe wood breaking
* Ring of Gathering can now revive pets/minions that have died while wearing the ring by animating the ring in an Animator
* Added chat message when player has not enough XP to use the Ring of Gathering
* Octine Arrows can now set blocks on fire like Octine Ingots
* Added new plants to Flower Power Advancement
* Added new Disgracing the Dead Advancement
* Reworked Ring of Recruitment to be much more useful
  + No longer constantly uses XP, but instead when a recruited mob attacks or is being healed by the ring
  + Number of recruited mobs is now limited based on health and attack damage of recruited mobs
  + Recruited mobs can no longer accidentally be damaged by the controlling player
  + Amulets & Amulet Slot can be equipped on recruited mobs
  + Improved overlay effect
* Updated Japanese localization. Thanks tento64!

### Bug Fixes

* Fixed Rotten Planks recipe not working properly
* Fixed Greataxes not dealing extra damage to Spirit Tree Faces and Living Roots
* Fixed Greataxes not disabling shields
* Fixed Ancient Battle Axe still able to break blocks even when fully damaged
* Fixed some mob AIs not able to disable shields
* Fixed some shader effects breaking when world is very old
* Fixed Basilisk Arrows working on bosses with full duration
* Fixed crash with bonemeal event
* Fixed Animator slot ghost items not rendering
* Fixed Animator running state not synced when items not inserted by player
