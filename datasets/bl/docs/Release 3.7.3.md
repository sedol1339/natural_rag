# Release 3.7.3

**Release 3.7.3** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on March 22, 2021. It remastered several music tracks, added subtitles, and made a few other changes and bug fixes.

Changelog
---------

### Additions

* Added subtitles

### Changes

* Several Music tracks have been remastered
* Aspect can now be directly squeezed out of Aspectrus Fruit into vials in the Mortar with greater speed and yield compared to infusing them

### Bug Fixes

* Fixed a potential memory leak and location data is now cached properly to reduce disk accesses while loading worlds
