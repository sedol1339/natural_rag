# Dry Bark

[Aside block] Dry Bark
----------------------

### Type

Material

### Tooltip

Used in: Mortar, Compost Bin

### Renewable

Yes

### Stackable

Yes (64)

### Compostable

Yes

### ID

items\_misc *(data value 13)*

**Dry Bark** is an item primarily used in making paper, or for composting.

It is considered a Plant Item, and can be ground into Ground Weedwood Bark.

Uses
------

### Compostability
Dry Bark can be composted in a Compost Bin. It has a compost process time of 10 minutes and a compost amount of 30, making it one of the most effective items to compost. Visit the Compost page for more information regarding the composting mechanic.
