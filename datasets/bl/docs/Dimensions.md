# Dimensions

The Betweenlands mod currently adds only one new dimension, The Betweenlands. However, there are a couple others planned for the future, along with minor changes and lore additions to The Overworld and The Nether.

The following is a list of dimensions, along with a brief description or list of changes the mod brings to them.

Contents
--------

* 1 Mod-Specific
* 2 Vanilla
* 3 Planned
* 4 History

Mod-Specific
--------------

* The Betweenlands - The main worldspace of the mod.

Vanilla
---------

* The Nether - No changes currently, only lore implications.
* The Overworld - Added Druid Circles to swamps.

Planned
---------

* The Prime Wight's Domain - The arena for the upcoming Prime Wight boss battle.
* The Snail Realm - A planned Easter Egg dimension.

History
---------

* **Beta 1.0.0**: Introduced The Betweenlands.
