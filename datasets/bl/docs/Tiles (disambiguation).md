# Tiles (disambiguation)

"Tiles" refers to many different blocks. This disambiguation page lists all tile variants added in the mod.

Blocks
--------

### **Manufactured**
* Betweenstone Tiles: Decorative tile blocks for Betweenstone.
* Cragrock Tiles: Decorative tile blocks for Cragrock.
* Limestone Tiles: Decorative tile blocks for Limestone.
* Mud Tiles: Decorative mud-based tile blocks.
* Pitstone Tiles: Decorative tile blocks for Pitstone.

### **Generated**
* Glowing Betweenstone Tile: A decorative light source found in the Wight Fortress.
* Weak Betweenstone Tiles: Blocks that break when walked upon, found mainly in the Wight Fortress.
* Broken Mud Tiles: A block that crumbles into Stagnant Water when walked on, found in the Sludgeon.
* Decayed Mud Tiles: Blocks that are slippery to walk on, found in the Sludgeon.
* Mud Tiles Spike Trap: Blocks that house deadly spikes, found in the Sludgeon.

### **Unimplemented**
* Dungeon Tile: A decorative block found in Ancient Temples.
