# Planks (disambiguation)

"Planks" refers to several different wood-based blocks. This disambiguation page lists all plank block variants added in the mod.

Blocks
--------

### **Manufactured**
* Giant Root Planks: Planks crafted with Giant Root blocks.
* Hearthgrove Planks: Planks crafted with Hearthgrove Logs.
* Nibbletwig Planks: Planks crafted with Nibbletwig Logs.
* Rotten Planks: Planks crafted with Rotten Bark.
* Rubber Tree Planks: Planks crafted with Rubber Tree Logs.
* Weedwood Planks: Planks crafted with Weedwood or Weedwood Logs.
