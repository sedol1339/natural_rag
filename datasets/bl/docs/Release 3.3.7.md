# Release 3.3.7

**Release 3.3.7** was an update to the Betweenlands for Minecraft 1.12.2 that was released on April 10, 2018. It added more configuration options, as well as some changes and bug fixes.

Changelog
---------

### Additions and Changes

* Added more config entries for Rotten Food, Tainted Potions, Fire Tools, Fertilizers, Tool Weakness, and Damp Torches
* Added localization for empty Rotten Food/Tainted Potion
* Added Russian localization by LIL\_SUZEE
* Weedwood Bushes are now farmable
* Respawn point is now changed if current respawn is invalid (e.g. in water)
* Improved Ring of Recruitment AI compatibility
* Increased Portal check radius
* Increased maximum Portal link distance to 1500 blocks

### Bug Fixes

* Workaround for Intel graphics driver crash
* Fixed some block models causing crash when non-forge lighting pipeline is used
* Fixed double tall underwater plant stalks not breaking when flower is broken
* Fixed swat shield advancement ConcurrentModificationException crash
* Fixed Mortars not saving their rotation
* Fixed "Give me that" advancement triggering on Swamp Talisman Pieces
* Fixed changing translation causing crash when opening Herblore Book
