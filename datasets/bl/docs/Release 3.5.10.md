# Release 3.5.10

**Release 3.5.10** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on April 5, 2020. It added a cooldown to the new Ring of Recruitment ability and fixed a couple of bugs.

Changelog
---------

### Changes

* Added short cooldown to Ring of Recruitment shield generate ability

Bug Fixes
---------

* Fixed Primordial Malevolence crash
* Fixed Ring of Recruitment shield generate ability being able to damage non-recruited enemies
