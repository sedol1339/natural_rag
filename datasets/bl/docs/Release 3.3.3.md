# Release 3.3.3

**Release 3.3.3** was an update to the Betweenlands for Minecraft 1.12.2 that was released on March 15, 2018. It added a portal overlay and some bug fixes.

Changelog
---------

### Changes

* Added overlay when in portal

### Fixes

* Fixed Herblore Book pages sometimes overlapping
* Fixed Wight causing client crash on MP when it tries to possess a player in specific circumstances
* Fixed portal sometimes dealing damage and incorrectly unlocking caverns advancement
* Fixed portal swirl going crazy for a short time after going through portal
