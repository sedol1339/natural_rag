# Release 3.5.2

**Release 3.5.2** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on November 14, 2019. It added a config option and fixed several bugs.

Changelog
---------

### Additions

* Added config for decay in other dimensions and to make other foods be able to heal decay
* Added block breaking overlay to Gecko Cage

### Bug Fixes

* Fixed crash with Iguana Tweaks
* Fixed Edge Leaves not having colour in inventory
* Fixed Dark Druids not attacking riding entities
* Fixed empty fluid vial causing crash with Forestry JEI Plugin
* Fixed tooltip incompatibility with Tinkers Construct
* Fixed Compost Bin dropping 64x the items it contains
* Fixed Ring of Dispersion overlay clearing depth, causing some shader incompatibilities
* Workaround for Multimine making Gecko Cages spawn Gecko twice
