# Alpha 3.2.1

*This is a snapshot of the upcoming 1.12.2 update. This is not the final version! Changes are to be expected and some things may not be fully implemented yet.*

**Alpha 3.2.1** was a snapshot hotfix update to the **Betweenlands** for Minecraft 1.12.2 that was released on December 15, 2017. It fixed issues with regular Torches becoming damp in dimensions other than the Betweenlands.

Changelog
---------

* Fixed Torches always turning into Damp Torches regardless of dimension
* Fixed modded Torches that extend BlockTorch not turning into Damp Torches
