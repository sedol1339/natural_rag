# Release 3.6.1

**Release 3.6.1** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on May 20, 2020. It made some adjustments to content from the previous version and fixed some bugs.

Changelog
---------

### Additions and Changes

* Added Chiromaw Barbs to Chiromaw Matriarch loot table
* Added new arrows to Arrow Arsenal advancement
* Greebling Riders now target the Chiromaw Matriarch
* Chiromaw Matriarch is now immune to petrify effect
* Gem Singer now only uses durability when marking a gem, but has slightly less max durability
* Gem Singer is now more reliable at marking gems

### Bug Fixes

* Fixed Greebling Riders hanging from ceilings
* Fixed Tamed Chiromaws targetting each other
* Fixed some structure loot pools not working until reloaded
* Fixed infinite Tamed Chiromaw double jumps
* Fixed Greataxes breaking blocks on rightclick swing
* Fixed Ancient Greatsword extended reach only working for default keybind
