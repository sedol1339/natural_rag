# Release 3.4.5

**Release 3.4.5** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on March 4, 2019. It added some equipment HUD position options and fixed some crucial bugs.

Changelog
---------

### Changes

* Added options to change position of equipment HUD
* Added a cooldown to Primordial Malevolence's deflection damage

### Bug Fixes

* Fixed fluids in Betweenlands Buckets having incorrect names
* Fixed incorrect fluid amounts in JEI Purifier recipes. Thanks sk2048!
* Fixed crash when Mud Brick Shingle Roof is picked up with gravity gun
* Fixed Primordial Malevolence pushing players away quickly when a damage source is deflected continuously
