# Release 3.4.2

**Release 3.4.2** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on December 29, 2018. It added a few minor changes and some bug fixes.

Changelog
---------

### Changes

* Underwater Roots can now be placed with the normal item
* Seasonal Events are now turned off when disabled in the config

### Bug Fixes

* Fixed being stuck on Harlequin Toad when shift right clicking
* Fixed getting items in inventories with a null side
* Fixed Ground Roots Herblore entry
* Fixed Silk Touch giving invalid block for Hearthgrove Logs
* Fixed some errors in the Chinese translation
