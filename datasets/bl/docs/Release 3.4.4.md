# Release 3.4.4

**Release 3.4.4** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on February 7, 2019. It added some Bukkit server compatibility and bug fixes.

Changelog
---------

### Changes

* Improved zh\_tw localization. Thanks mcBegins2Snow!
* Added compatibility with Bukkit type servers Atom and CatServer. Thanks Luohuayu!

### Bug Fixes

* Minor texture correction for Squared Amate Paper Pane
* Fixed Betweenlands arrows applying effects even when deflected (e.g. by Primordial Malevolence's shields)
* Fixed Thatch blocks letting through rain
