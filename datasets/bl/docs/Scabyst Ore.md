Scabyst Ore in-game

**Scabyst Ore** is an ore that drops Scabyst, which can be used to prevent Corrosion on equipment and for other uses.

Obtaining
-----------

### Generation
Scabyst Ore generates uncommonly underground between Y-layers 45 and 0 within Pitstone, in the Lake Cavern layer.

### Drops
Scabyst Ore can drop rarely from Boulder Sprites.

Drops
-------

Scabyst Ore drops 1-3 Scabyst and 4-10 XP when broken. The block itself must be broken with a Silk Touch tool to be collected.
