# Beta 2.0.4

**Beta 2.0.4** was an update to the **Betweenlands** for Minecraft 1.10.2 that was released on June 30, 2017. It added missing keybind lang and a few bug fixes.

Changelog
-----------

* Added missing lang for keybinds
* Fixed buckets using incorrect fluid localization
* Fixed Lurker Skin Pouch keybind not working
* Fixed Ring of Power particles always spawning
* Disabled Animator recipes
