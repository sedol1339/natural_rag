# Release 3.5.7

**Release 3.5.7** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on February 24, 2020. It added a new gamerule and fixed some bugs.

Changelog
---------

### Additions and Changes

* Added es\_ES localization, thanks NathanielFreeman!
* Added gamerule to disable timed Events changing state

Bug Fixes
---------

* Fixed Greatsword/Battle Axe type weapons' AoE damage not working properly
* Fixed Rubber Tap fluid texture
* Fixed Ring of Recruitment crash
