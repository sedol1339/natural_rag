# Release 3.5.9

**Release 3.5.9** was a very small update to the Betweenlands for Minecraft 1.12.2 that was released on April 2, 2020. It added new functionality to the Ring of Recruitment and fixed a spawning bug.

Changelog
---------

### Changes

* Added new shield ability to Ring of Recruitment to be more fitting as a drop from the Primordial Malevolence

Bug Fixes
---------

* Fixed Harlequin Toad not spawning properly
