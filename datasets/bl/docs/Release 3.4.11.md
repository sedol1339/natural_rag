# Release 3.4.11

**Release 3.4.11** was a small update to the Betweenlands for Minecraft 1.12.2 that was released on August 21, 2019. It was known as **The Sushi Update** as it added...well...Sushi.

Changelog
---------

### Additions

* Added Sushi
* Happy Birthday Jacob

### Bug Fixes

* Fixed some items not showing correctly in JEI
* Added Tarred Weedwood Rowboat info and Druid Circle reactivation to JEI
* Added missed items to Getting Harder Wood and Shield Wall advancements
