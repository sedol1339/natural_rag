Sulfur Ore in-game

**Sulfur Ore** is an ore that drops Sulfur, which can be used as a fuel source and for other uses.

Obtaining
-----------

### Generation
Sulfur Ore generates commonly underground between Y-layers 128 and 45 within Betweenstone, largely in the Cavern layer.

### Drops
Sulfur Ore can drop uncommonly from Boulder Sprites.

Properties
------------

### Particles
Small yellow fume particles constantly emanate from Sulfur Ore.

Drops
-------

Sulfur Ore drops Sulfur and 0-2 XP when broken. The block itself must be broken with a Silk Touch tool to be collected.
