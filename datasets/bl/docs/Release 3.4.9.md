# Release 3.4.9

**Release 3.4.9** was a very small update to the Betweenlands for Minecraft 1.12.2 that was released on June 17, 2019. It fixed a few rendering bugs as well as a mod incompatability.

Changelog
---------

### Bug Fixes

* Fixed Living Weedwood Shield causing a rendering bug in inventory
* Fixed Weedwood Chest item renderer
* Fixed an incompatbility with Dynamic Stealth. Thanks Laike-Endaril!
