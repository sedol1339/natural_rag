Movement
----------

Like Weedwood Chip Paths, Weedwood Walkways provide Speed II () to players that walk over them. This makes them very useful for traversing terrain more quickly.

Fuel
------

Weedwood Walkways can be used as fuel for the Sulfur Furnace or Dual Sulfur Furnace, smelting 1.5 items per path block.
