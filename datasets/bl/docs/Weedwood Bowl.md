# Weedwood Bowl

[Aside block] Weedwood Bowl
---------------------------

### Type

Material

### Renewable

Yes

### Stackable

Yes (64)

### Compostable

No

### ID

items\_misc *(data value 22)*

**Weedwood Bowls** are items used in making a few foods, such as mire scramble and weeping blue petal salad, as well as collecting drinkable brews and dyes.

### Drops
Weedwood Bowls can drop from looting Greebling Coracles.

### Emptying
A Weedwood Bowl is returned to the player upon:

* Eating Weeping Blue Petal Salad or Mire Scramble, or drinking any Drinkable Brew.
* Using any Dye in a crafting recipe.

Uses
------

### Collection
A Weedwood Bowl can be right-clicked on a Steeping Pot containing any Drinkable Brew or Dye fluid to collect it in its item form. Each bowl collects 250 mB worth of fluid, meaning that every steeping process yields up to 4 bowls.

### Crafting Uses
| Result | Ingredients | Recipe |
| --- | --- | --- |

| Mire Scramble | Shapeless: **Weedwood Bowl** + Cooked Mire Snail Egg + Yellow Dotted Fungus |  |
| Olmlette Mixture | Shapeless: **Weedwood Bowl** (is not consumed) + Flathead Mushroom (item) + Black Hat Mushroom (item) + Olm Egg Cluster (2) + Weedwood Stick (is not consumed) |  |
