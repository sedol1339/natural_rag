# Purifying

The following is a list of all recipes that utilize the Purifier, sorted into categories in alphabetical order.

Every recipe requires Sulfur and a portion of Swamp Water inside the Purifier in the purifying process.

For more information on how Purifying works, visit the Purifier page.
