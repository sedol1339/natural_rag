# Release 3.3.8

**Release 3.3.8** was an update to the Betweenlands for Minecraft 1.12.2 that was released on April 25, 2018. It added configuration and command changes, as well as some other changes and bug fixes.

Changelog
---------

### Additions and Changes

* Added config option to disable shaders when not in dimension
* Added config option to start players in the dimension
* Added config for maximum Portal link distance
* Added config and warning tooltip for Caving Rope despawn time
* Replaced Octine Ingot with Bulb-Capped Mushrooms in the Caving Rope recipe
* Changed Ring of Summoning to only target hostiles
* Added /setDecay command
* /aspectDiscovery command now requires target player
* Updated minimum Forge requirement to 14.23.2.2643
* Updated zh\_CN localization

### Bug Fixes

* Fixed world locations sometimes not syncing properly when teleporting
* Fixed Life Crystals always dropping with 1 damage
* Fixed Weedwood Crafting Table rendering items mirrored
* Fixed decay food not filling up bar if last item in stack is eaten
* Fixed bug with Caving Rope, significantly increasing performance
* Fixed Valonite Shield causing crash when charging into each other
* Fixed Loot Tables not initialized during preInit
